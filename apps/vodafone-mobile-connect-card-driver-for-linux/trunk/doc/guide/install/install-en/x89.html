2. Installation

2.1. Dependencies

The bundled version needs the next dependencies to work correctly, most of them are satisfied by the most of the modern GNU/Linux distributions:

  • Linux Kernel (2.6.20 or superior recommended for most of the cards) with usbserial and udev support. Nozomi module driver also needed for Option GlobeTrotter 3G+ EMEA card

  • dbus 1.1 or superior.

  • Hardware Abstraction Layer HAL.

  • X Window System.

2.2. Installation

  1. Download the installer from Project page

  2. Run the installer from a terminal as user root.

  3. The installer will ask you for the users allowed to run the application. It automatizes the adding of inserted users in the needed groups. Introduce them space separated.

  4. When installation has been completed, restart your session to make the system refresh your user groups.
