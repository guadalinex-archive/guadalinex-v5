Recogida de datos para Hermes.

Introducción

Hermes es el notificador y detector de cambios en el hardware de Guadalinex v5. Es el encargado de avisar al usuario de la detección de dispositivos conectados al sistema, así como de instalar los paquetes necesarios para que esos dispositivos funcionen.

Para el correcto funcionamiento de Hermes, este debe conocer información sobre los distintos dispositivos que se pueden conectar al sistema. Para la recogida de esta informacion se ha creado el "Informe sobre nuevos dispositivos".

¿Cómo obtener la información?

El software necesario para recoger información sobre dispositivos está incluído en Guadalinex V5. Este software no es más que un asistente embebido en el sistema de notificación de cambios en el hardware: HERMES. La recogida de datos consta de tres pasos: inicio, obtención de datos, y fin.

El paso realmente útil es el segundo, que es el momento en el que hermes obtiene la información del dispositivo. Es MUY IMPORTANTE para el proceso de obtención de datos seguir bien las indicaciones de este paso, a saber:

  1. Conectar el dispositivo. Es en este paso donde debe conectar el dispositivo al sistema (al puerto usb, firewire, o donde proceda).
  2. Esperar unos 10 segundos. Durante este período de tiempo es cuando hermes obtiene los datos del dispositivo. Puede variar en función del tipo de dispositivo que se esté detectando. Por regla general, este paso estará acompañado de mensajes de notificación del sistema (abajo y a la derecha, junto al panel).
  3. Desconectar el dispositivo. Al contrario que en el paso 1, el dispositivo debe quedar desconectado del sistema.

Tras concluir dicho asistente, debe aparecer un nuevo fichero de texto en el escritorio, el cual contiene toda la información que Hermes necesita para poder administrar correctamente el dispositivo en cuestión.

¿Cómo hacernos llegar la información?

Tras la ejecución del "Informe sobre nuevos dispositivos" podemos encontrar un fichero en el escritorio que contiene toda información necesaria para que Hermes pueda identificar el nuevo dispositivo.

Es necesario hacer llegar esa información al equipo de desarrollo de Hermes. Para ello se ha habilitado la siguiente cuenta de correo: hermes@guadalinex.org.

Así pues, una vez tengamos el fichero con la información sobre el dispositivo en el escritorio, basta con enviar un correo a esa dirección adjuntándolo.

Para más información puede consultar la siguiente dirección: http://www.guadalinex.org/guadapedia/index.php/Hermes
