Desinstalar Escritorio movistar

 

«« AnteriorSiguiente »»


Para desinstalar la aplicación correctamente necesitas entrar en el sistema operativo como usuario root (administrador).

Al igual que el proceso de instalación, el proceso de desinstalación es muy sencillo. Puedes utilizar cualquiera de las opciones que se expresan a continuación:

  • Para desinstalar la aplicación desde Fedora Core 9 sigue los siguientes pasos:

    • Desde el entorno gráfico, pulsa en Aplicaciones >> Agregar/Quitar Software. A continuación aparecerá la ventana del Administrador de Paquetes y desde ahí podrás localizar la aplicación y desinstalarla.

    • Desde la línea de comandos, debes ejecutar la siguiente instrucción:

yum remove escritorio-movistar

 

  • Para desinstalar la aplicación desde SUSE Linux 11 sigue los siguientes pasos:

    • Desde el entorno gráfico, abre la herramienta YaST, después en la pestaña Software pulsa en Instalar/Desintalar Software. En la ventana que aparece, localiza la aplicación y podrás desinstalarla.

    • Desde la línea de comandos, debes ejecutar la siguiente instrucción:

zypper remove escritorio-movistar

 

  • Para desinstalar la aplicación desde Ubuntu 8.04 sigue los siguientes pasos:
    • Pulsa en Sistema >> Administración >> Gestor de paquetes Synaptics. A continuación aparecerá la ventana del Administrador de Paquetes y desde ahí podrás localizar la aplicación y desinstalarla.
    • Desde la línea de comandos, debes ejecutar la siguiente instrucción:

apt-get remove escritorio-movistar
