Instalar el software de Escritorio movistar

Componentes de la instalación

«« AnteriorSiguiente »»


La instalación de Escritorio movistar incluye los siguientes servicios:

  • movistar Internet.

Se trata del acceso general a Internet. Tienes la opción de abrir el navegador configurado como predeterminado en tu equipo tras establecer la conexión, abriendo el sitio web que hayas elegido (el configurado inicialmente es: http://www.movistar.es).

  • Mensajes.

Envía mensajes SMS y MMS desde tu ordenador mediante el servicio de Mensajería Web de movistar. Este servicio precisa de tus datos de identificación de usuario (Teléfono y Clave).

  • Correo Móvil.

El servicio de Correo Móvil establece una conexión a la red movistar e inicia el servicio de Correo Web de movistar. Este servicio precisa tus datos de identificación del usuario (Teléfono y Clave). La contraseña se envía siempre de forma cifrada por razones de seguridad.

  • Zonas WiFi de Telefónica.

Con cualquiera de las Tarifas Banda Ancha Móvil de calidad Premium podrás conectarte en las Zonas WiFi de Telefónica gratuitamente y de forma ilimitada.

  • movistar Intranet.

Este servicio está concebido para la realización de conexiones a la Intranet corporativa de tu empresa. De forma predeterminada, este servicio no está configurado, por lo que deberás ponerte en contacto con el administrador de la red de tu empresa para que te facilite los datos de conexión.
