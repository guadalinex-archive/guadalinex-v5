Resolución de problemas
 

La instalación/desinstalación no se ejecuta en modo gráfico si utilizo SUSE Linux 10.3

  • La configuración predeterminada de esta distribución no permite ejecutar aplicaciones en modo gráfico como root desde otro usuario. Por ello, para poder ejecutar en modo gráfico el proceso de instalación/desinstalación de la aplicación Escritorio movistar en la distribución SUSE Linux 10.3, es necesario que accedas al sistema como usuario root.

|Volver arriba|

 

No puedo ejecutar la aplicación correctamente

  • Para un correcto funcionamiento de la aplicación Escritorio movistar, es necesario tener instalado el escritorio Gnome.

|Volver arriba|

 

No puedo conectarme

  • Comprueba que tu dispositivo (ya sea una Tarjeta o Módem USB Internet Móvil o un Teléfono) y el ordenador están conectados entre sí y que existe comunicación entre ambos.
  • Comprueba que tienes establecidas correctamente las opciones de Configuración del servicio que deseas iniciar.
  • Si el problema persiste, contacta con el servicio de Atención en L ínea en la dirección URL http://www.atencionenlinea.movistar.es o puedes llamar de forma gratuita al 609 desde teléfonos movistar o al 1485 desde cualquier otro teléfono. Es muy importante que antes de ponerte en contacto con el servicio de Atención en Línea repitas todos los pasos que han provocado tu duda.

|Volver arriba|

 

No puedo navegar si utilizo SUSE Linux 10.3 

  • Si la aplicación NetworkManager se encuentra en ejecución, pueden aparecer problemas en la navegabilidad (el navegador arranca en modo "Trabajar sin conexión"). Para evitar este problema es aconsejable que cierres esta aplicación; para ello accede al menú Aplicaciones >> Sistema >> YasT >> Dispositivos de red >> Tarjetas de red y selecciona la opción Método tradicional con ifup. A continuación pulsa en Siguiente y después en Finalizar. Para que esta modificación surta efecto es aconsejable reiniciar el equipo.

|Volver arriba|

 

Algunas aplicaciones no se conectan a Internet correctamente cuando utilizo el Escritorio movistar

  • La conexión movistar Internet  que incluye el Escritorio movistar optimiza el tráfico de información para conseguir una navegación más fluida y podría llegar a afectar al funcionamiento de alguna aplicación.

  • Si detectas un problema de funcionamiento con aplicaciones que accedan a Internet mientras utilizas el Escritorio movistar, prueba a utilizar la conexión movistar Internet directo.

  • Consulta la sección Configuración >> Conexiones para obtener más información al respecto.
