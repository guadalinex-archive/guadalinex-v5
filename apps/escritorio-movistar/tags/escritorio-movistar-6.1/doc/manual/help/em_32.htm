Conexión del Teléfono con Bluetooth
 

Para que un Teléfono con bluetooth enlace correctamente con un portátil hay que tener en consideración las siguientes pautas:

 

  • Primero debes abrir un Terminal y arrancar el servicio bluetooth con el comando start. Habitualmente este servicio se encuentra en /etc/init.d/bluetooth.
  • Para comprobar que el servicio bluetooth funciona correctamente, puedes utilizar el comando hciconfig –a, que devuelve la dirección MAC local.
  • A continuación, para encontrar el Teléfono con el que quieres establecer la conexión, utiliza el comando hcitool scan, el cual muestra todos los dispositivos bluetooth que están al alcance (su dirección MAC, nombre).
  • Puedes interrogar al dispositivo y consultar sobre el establecimiento de la llamada a través del comando sdptool search DUN.
  • Finalmente,  debes establecer el canal con el comando rfcomm bind * @MAC channel, donde, “*” es el número del dispositivo rfcomm y “channel” corresponde a la información devuelta por el comando anterior.
  • En este momento se te pedirá que introduzcas el PIN de la asociación de dispositivos en el Teléfono (no lo confundas con el PIN de la tarjeta SIM). Para confirmar la correcta asociación puedes ejecutar el comando rfcomm show.
  • Puedes eliminar el vínculo con el comando rfcomm release rfcomm*

 

Asegúrate de que en Configuración >> Mis dispositivos >> Otro dispositivo está seleccionado como dispositivo  GPRS/3G el Teléfono que desees utilizar.
