Cómo navegar por Internet utilizando Escritorio movistar.

  1. Abre la aplicación de Escritorio movistar en tu ordenador, pulsando sobre el icono correspondiente.

  2. En el caso de que uses Escritorio movistar con una Tarjeta o Módem USB Internet Móvil, ésta se seleccionará automáticamente, sino selecciónala desde Configuración >> Mis dispositivos >> Tarjeta o Módem USB Internet Móvil. A continuación te pedirá el código PIN de tu tarjeta SIM, si éste se encuentra activado; una vez introducido, pulsa Aceptar y espera a que el indicador de cobertura de la red del Visor de Información se ponga en verde.

  3. En el caso de que accedas mediante un Teléfono, si no hay ninguno seleccionado, selecciónalo a través de Configuración >> Mis dispositivos >> Teléfono móvil. Para saber cómo configurar y activar estas opciones puedes consultar la sección Instalación del hardware.

  4. Por último pulsa el botón movistar Internet de la zona de servicios y se abrirá el navegador web de tu equipo con la página de inicio del mismo.

Puedes consultar el apartado Resolución de problemas si has encontrado alguna dificultad en este proceso.

Cómo enviar un correo electrónico utilizando Escritorio movistar.

  1. Abre la aplicación de Escritorio movistar en tu ordenador, pulsando sobre el icono correspondiente.

  2. Si no hay dispositivo de conexión seleccionado, puedes seleccionarlo pulsando en Configuración >> Mis dispositivos. Consulta el punto Cómo navegar por Internet utilizando Escritorio movistar para saber cómo seleccionar correctamente los dispositivos de conexión.

  3. A continuación, pulsa en el botón Correo Móvil de la zona de servicios. Si es la primera vez que accedes al servicio tendrás que introducir Teléfono y Clave.

Puedes consultar el apartado Resolución de problemas si has encontrado alguna dificultad en este proceso.

Cómo enviar un mensaje de texto o mensaje multimedia utilizando Escritorio movistar.

  1. Abre la aplicación de Escritorio movistar en tu dispositivo, pulsando sobre el icono correspondiente.

  2. Si no hay dispositivo de conexión seleccionado, puedes seleccionarlo pulsando en Configuración >> Mis dispositivos. Consulta el punto Cómo navegar por Internet utilizando Escritorio movistar para saber cómo seleccionar correctamente los dispositivos de conexión.

  3. Si es la primera vez que accedes al servicio tendrás que introducir Teléfono y Clave

  4. Pulsa el botón Mensajes de la zona de servicios y accederás a la página web del servicio solicitado donde se te indicará la forma en la que puedes enviar tu mensaje.

 

Recuerda que si todavía no eres usuario de los servicios movistar o has olvidado tu clave de acceso, solicita tu clave enviando con tu móvil un mensaje de texto gratuito al 770 cuyo texto sea la clave que desees. Ejemplo, envía Azules al 770 para que ésa sea tu clave de acceso.

Puedes consultar el apartado Resolución de problemas si has encontrado alguna dificultad en este proceso.
