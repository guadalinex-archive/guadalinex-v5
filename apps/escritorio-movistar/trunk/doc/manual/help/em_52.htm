Correo Móvil
 

 

Si todavía no eres usuario de los servicios movistar o has olvidado tu clave de acceso, solicita tu clave enviando con tu móvil un mensaje de texto gratuito al 770 cuyo texto sea la clave que desees. Ejemplo, envía Azules al 770 para que sea tu clave de acceso.

El servicio de Correo Móvil establece una conexión a Internet y abre el servicio de Correo Móvil usando la interfaz web o, de forma opcional, tu cliente de correo predeterminado, según lo que tengas seleccionado en el apartado Configuración >> Servicios >> Correo Móvil.

El servicio Correo Móvil, cuando se utiliza la interfaz web, precisa de tus datos de autenticación (Teléfono y Clave) para el acceso a tu página de usuario; estos datos puedes introducirlos desde Configuración >> Datos de usuario.

Figura 14. Servicio Correo Móvil.

 

En el momento de redactar este documento el navegador compatible con este servicio es Mozilla Firefox y Epiphany.
