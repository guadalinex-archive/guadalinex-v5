La aplicación Escritorio movistar no se ejecuta correctamente

  • Para un correcto funcionamiento de la aplicación Escritorio movistar, es necesario tener instalado el escritorio Gnome.

    |volver arriba|

     

El componente Mobile Manager no se encuentra disponible

  • Es posible que al ejecutar Escritorio movistar se muestre un mensaje relativo a que el componente Mobile Manager no se encuentra disponible. En este caso deberás reiniciar el equipo o bien arrancar el demonio a través de la línea de comandos utilizando la siguiente instrucción: /etc/init.d/mobilemanager start
  • Observa que es necesario ser usuario root para poder llevar a cabo esta acción. Una vez levantado el demonio ya podrás ejecutar la aplicación de forma correcta.
|volver arriba|

Algunas aplicaciones no se conectan a Internet correctamente cuando utilizo Escritorio movistar

  • La conexión movistar Internet que incluye Escritorio movistar optimiza el tráfico de información para conseguir una navegación más fluida y podría llegar a afectar al funcionamiento de alguna aplicación.

  • Si detectas un problema de funcionamiento con aplicaciones que accedan a Internet mientras utilizas Escritorio movistar, prueba a utilizar la conexión movistar Internet directo.

  • Consulta la sección Configuración >> Conexiones para obtener más información al respecto.

|volver arriba|

 

No puedo conectarme a Internet

  • Comprueba que tu dispositivo (ya sea una Tarjeta o Módem USB Internet Móvil o un Teléfono) y el ordenador están conectados entre sí y que existe comunicación entre ambos.
  • Comprueba que tienes establecidas correctamente las opciones de Configuración del servicio que deseas iniciar.
  • Si el problema persiste, contacta con el servicio de Atención en Línea en la dirección URL http://www.atencionenlinea.movistar.es o puedes llamar de forma gratuita al 609 desde teléfonos movistar o al 1485 desde cualquier otro teléfono. Es muy importante que antes de ponerte en contacto con el servicio de Atención en Línea repitas todos los pasos que han provocado tu duda.
|volver arriba|

 

No puedo navegar si utilizo SUSE Linux 11

  • Si la aplicación NetworkManager se encuentra en ejecución, pueden aparecer problemas en la navegabilidad (el navegador arranca en modo Trabajar sin conexión).
  • Para evitar este problema es aconsejable que cierres esta aplicación; para ello accede al menú Aplicaciones >> Sistema >> YasT >> Dispositivos de red >> Tarjetas de red y selecciona la opción Método tradicional con ifup. A continuación pulsa en Siguiente y después en Finalizar. Para que esta modificación surta efecto es aconsejable que reinicies el equipo.
    |volver arriba|

 

Mi navegador siempre se inicia en modo fuera de línea con Fedora Core 9

  • Este comportamiento es debido a la aplicación NetworkManager, para habilitar el navegador deberás acceder al menú Archivo del mismo y deshabilitar la opción Trabajar sin conexión.
