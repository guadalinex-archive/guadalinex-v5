Mensajes
 
Si todavía no eres usuario de los servicios movistar o has olvidado tu clave de acceso, solicita tu clave enviando con tu móvil un mensaje de texto gratuito al 770 cuyo texto sea la clave que desees. Ejemplo, envía Azules al 770 para que sea tu clave de acceso.

Este servicio te proporciona acceso a la Mensajería Web de movistar, desde donde podrás enviar mensajes de texto (SMS) y mensajes multimedia (MMS). Este servicio precisa de tus datos de autenticación (Teléfono y Clave), que puedes configurar en Configuración >> Datos de usuario.


Figura 13. Servicio Mensajes.

 

En el momento de redactar este documento el navegador compatible con este servicio es Mozilla Firefox y Epiphany.
