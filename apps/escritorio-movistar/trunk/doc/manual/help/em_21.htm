Instalar el software de Escritorio movistar

Requisitos de instalación

«« AnteriorSiguiente »»


Las características mínimas con las que debe contar tu ordenador para ser compatible con el servicio Escritorio movistar 6.5 para Linux son las siguientes:

Hardware:

  • Ordenador PC o compatible con ranura PCMCIA para Tarjetas Internet Móvil o puerto USB 2.0 en caso de Módem Internet Móvil o puerto de conexión al Teléfono que desees utilizar (bluetooth, USB, puerto serie o infrarrojos).
  • Tarjeta SIM dada de alta en el servicio GPRS/3G de movistar. 
  • Tarjeta o Módem USB Internet Móvil.
  • Teléfono con conexión GPRS/3G.

Sistema Operativo:

  • Linux, distribuciones soportadas: Fedora Core 9 (con versión de Kernel 2.6.25.6-55 ), SUSE Linux 11 (con versión de Kernel 2.6.25.5-1.1) y Ubuntu 8.04 (con versión de Kernel 2.6.24-18).

 

Comprueba que tu Tarjeta o Módem USB Internet Móvil dispone de un firmware actualizado. Si no estuviera actualizado, podrás descargar las últimas versiones desde la dirección URL: http://www.movistar.es

Para utilizar el servicio Escritorio movistar debes ser cliente de Telefónica Móviles España y estar dado de  alta en el servicio GPRS/3G. Para comprobar que estás dado de alta en el servicio, puedes llamar al número de atención al cliente (609) de movistar.
